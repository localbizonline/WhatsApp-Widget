WhatsApp Widget

Add a branded WhatsApp chat widget to any website with a single script tag. No build step, no dependencies.

Minimal

Basic embed (WhatsApp only)

Just a phone number and brand name. That's all you need.

<script
  src="widget.js"
  data-phone="27787869161"
  data-brand="Acme Co"
></script>
Full

Full embed (logo + demo link + custom colors)

All options in one embed. Logo, custom color, booking link, custom messages.

<script
  src="widget.js"
  data-phone="27787869161"
  data-brand="ReachMax"
  data-logo="https://example.com/logo.png"
  data-color="#25D366"
  data-welcome="How can we help you today?"
  data-demo-url="https://cal.com/your-link"
  data-demo-label="Book a Free Demo"
  data-message="Hi! I'd like to know more."
></script>
